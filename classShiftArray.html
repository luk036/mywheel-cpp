Detailed Description

template<typename T>
class ShiftArray< T >

Array with shifted index access semantics.

Allows accessing elements of a container using indices that are offset by a configurable start value.

Template Parameters
TType of elements in the array.
* Original indices:  [0] [1] [2] [3] [4] [5]
* Array content:     [A] [B] [C] [D] [E] [F]
*
* After set_start(2):
*
* Accessible range:        [0] [1] [2] [3]  <-- These map to [C] [D] [E] [F]
* Shifted indices:         [2] [3] [4] [5]  <-- Internal indices
*
